Viewport based selection

属性: srcset \ sizes \ w \ src \ vm \ w \ h 在 img 标签上

srcset: 资源库

sizes: 条件

vw / vh: 视窗单位。

w: 描述器（指定作用），图片的规格

user agent 选择图片的条件: 1.sizes 2.有效的像素密度 3.w 描述器

最佳设备像素比 device pixel ratio = w / sizes 结果

Note: src 不参与资源选择，为兼容性而设置，当srcset设置了w描述时，新的user agents将忽略src。
